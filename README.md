# Zendesk Searcher Application

## Frontend setup
```
cd frontend-web
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

## Backend setup
```
cd backend
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Run tests
```
npm test
```
